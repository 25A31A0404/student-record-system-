# student record system
📌 Overview

The Student Record System is a simple console-based application written in C that helps manage student data efficiently. It allows users to store and view student information such as roll number, name, and marks.

This project is useful for beginners to understand concepts like structures, file handling, and menu-driven programs in C.

✨ Features
➕ Add new student records
📄 View all student records
💾 Store data using file handling (optional version)
📋 Simple and user-friendly menu interface
🛠️ Technologies Used
C Programming Language
Standard I/O functions (stdio.h)
File handling (fopen, fwrite, fread)
🚀 How to Run
Install a C compiler (like GCC)
Save the code in a file (e.g., main.c)
Open terminal/command prompt

Compile the program


Run the program


📖 How It Works
The program displays a menu with options
User selects an operation (add/view/exit)
Data is stored either in memory (array) or file
File version keeps records saved permanently
